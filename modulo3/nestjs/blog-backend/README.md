# Blog Backend (NestJS)

Proyecto base para el backend del blog con NestJS.

## Comandos sugeridos

```bash
nest generate module basics
nest generate controller basics
nest generate service basics
```

## Desarrollo

- `npm run start:dev` arranca el servidor en modo watch.
- `npm run build` compila a `dist/`.
